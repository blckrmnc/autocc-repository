How does it work?
[1] Select an image by pressing the "Select Image" button.
[2] Set your settings by typing values in the input box.
	(Optimal settings are from 0.5 to 1.5)
[3] After that, press the "Enhance Image" button.
[4] You will see a folder named "cc_folder" and it will be automatically saved there.
